# Cáscara Planificación

Web app estática hosteada en GitHub Pages.

## Estructura

```
.
├── index.html
├── css/styles.css
└── js/app.js
```

## Desarrollo local

Abrir `index.html` en el navegador, o servirlo con:

```bash
python3 -m http.server 8000
```

Luego ir a http://localhost:8000

## Deploy

Cada push a `main` se publica automáticamente vía GitHub Pages.
